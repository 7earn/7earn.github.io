# VMware 虚拟机安装

1、首先，打开虚拟机界面，然后点击 “创建新的虚拟机”

2、这里直接点击下一步，到这个界面，选择 “自定义（高级）”：

3、然后直接点击 “下一步” 直到这个界面，然后我们选择 “稍后安装操作系统”：

4、选择操作系统为 “Linux”，版本选择 “Ubunta 64 位”

5、虚拟机名称，看个人喜好，自己选择；安装的位置，也是自己选择的，在选择好后，就直接下一步。

6、在这根据自己电脑的配置，进行相应的选择

8、在这一步，同样是自己选择的，根据自己的电脑配置来分配，如果觉得自己的电脑配置比较高就可以适当地多分配一点。

9、对于网络的选择，就直接默认就行了

10、对于控制器类型和磁盘类型，按照它推荐的来（这个我也不是特别的清楚，如果想知道为什么的话，就可以在网上自己搜索一下）


11、 选择 “创建新虚拟磁盘”（看自己喜好吧，它上面有相应的介绍）

12、选择磁盘大小，根据自己的情况选择

13、选择磁盘文件储存的位置，自己选择

14、点击完成，新虚拟机创建完成

# Linux 镜像安装

在完成新的虚拟机的创建以后，就需安装镜像的安装了：
1、首先，点击虚拟机设置，然后再 CD/DVD 中选择 “使用 ISO 镜像文件”，选择自己镜像下载的位置，弄好以后，点击确定就行了。

2、上面还有一个图片，但是我忘记截图了，很无语，但是这不影响。（建议：在安装的时候要注意屏幕截图，它左下角就是截图，方便后面出现问题，寻找原因）
到了这一步，推荐选择的是英语安装，但是由于，我英语比较菜，所以我就选择汉语了。
2、这两步就不多说了，弄好以后就是在配置网络了。


3、这一步就是我们主机名称，可以自己选择

4、这一步可以不填，直接继续

5、这两步就是设置自己的用户名（我们登录 QQ 是一样，需要账户和密码，当我们弄好虚拟机以后，它会让我们进行登录，这个时候就需要账户和密码）
6、设置开机密码：

7、设置磁盘分区，作为新手的话，我建议选择整个磁盘（我也是小白，选的也是整个磁盘，不过我看网上的大佬说，选择整个磁盘，可能会使文件丢失，这样的情况对于我们而言有好有坏吧，因情况而定，主要还是看自己选择）





8、结束这些以后，就是等待它安装基本系统

9、这一步，如果没有的话，就按照它的来吧，然后在等待它的安装 （如果有的话，我认为还是填一下，挂代理，省的出现后面会遇见的问题，遇见的话我觉得还是很麻烦的，不过方法也是有很多种的）


10、这一步直接继续（这一步可能不会遇见，我创建的第一个虚拟机就没有遇见过这个情况）
11、这一步，一般就直接继续就好了（在这个选择安装软件的一定要勾选 Xfce，否则在进入虚拟机的时候会没有界面，也就是说和正常的用户界面有一定的区别，就很离谱）
13、遇见这种情况，我们不要慌，虽然我自己慌了一会，哈哈哈 (解决这个问题有很多种方法，具体还是要看自己的选择)
